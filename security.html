Security

The web bridge is a ciphertext transport layer. Messages are encrypted in the browser. Only the recipient device can decrypt.

End-to-end encrypted
Ciphertext-only storage
Device-held private keys
Rolling is constrained

Encryption boundary

Messages sent through setfeed.app are encrypted in the browser. The server never receives plaintext.

Send flow (high level)

Browser encrypts → policy validated → ciphertext stored → device fetches → decrypts locally
